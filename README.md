## üìå Features

- Step-by-step animations
- Adjustable array size and animation speed
- Supports multiple algorithms:
  - Bubble Sort
  - Insertion Sort
  - Selection Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
  - Radix Sort
  - Bucket Sort
  - Counting Sort
- Real-time comparison
- Performance evaluation
- Simple, responsive UI
